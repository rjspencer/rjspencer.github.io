Using the * Wildcard

You can use the * asterisk symbol as a wildcard character in your commands. The obvious reason to use a wildcard is when you only know part of what you are looking for, but you can also use it as a speed up. Instead of typing cd annual_reports_2014 you can just type cd *14 if you know it is the only folder that ends with 14. You can use this trick with files and folders and even during program calls like ruby 2* or sqlite3 *cat.
