ARRAY & OBJECT

All variables must be declared or they become global! var array = []; var array = [ 1, "cat" ]; array.push(-1200); array === [ 1, "cat", -1200 ]; array[2] === -1200; array[-2] === "cat";

Objects are similiar to Ruby Hashes, but cannot be accessed by index[num] var object = {}; var object = { 1: "Alaska", blue: orange", "The answer to life": 42, -100: 200 }; // key becomes a symbol with or without '' object["new_key"] = "New Value"; object === {1: "Alaska", blue: orange", The answer to life: 42, -100: 200, new_key: "New Value" };
